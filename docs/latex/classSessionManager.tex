\doxysubsection{Detailed Description}
Manages multiple simulation sessions and their lifecycle. 

The \doxylink{classSessionManager}{Session\+Manager} class provides centralized management of multiple simulation sessions within the MSFS \doxylink{namespaceDIS}{DIS} Bridge system. It handles session creation, termination, and cleanup operations while maintaining a collection of active sessions.

This class serves as the primary interface for session management, allowing the bridge system to maintain multiple concurrent connections or simulation instances. Each session is managed through a \doxylink{classSimSession}{Sim\+Session} object that handles the specific details of that connection.

\begin{DoxyNote}{Note}
The \doxylink{classSessionManager}{Session\+Manager} uses RAII principles to ensure proper cleanup of all managed sessions when the manager is destroyed. 
\end{DoxyNote}
