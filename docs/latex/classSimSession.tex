\doxysubsection{Detailed Description}
Individual simulation session managing one Sim\+Connect connection. 

The \doxylink{classSimSession}{Sim\+Session} class represents a single connection to a Microsoft Flight Simulator instance via Sim\+Connect. Each session runs independently in its own thread and manages the full lifecycle of data collection from MSFS.

This class handles\+:
\begin{DoxyItemize}
\item Establishing and maintaining Sim\+Connect connection
\item Defining and requesting flight data variables
\item Processing incoming simulation data
\item Managing simulation state (running/paused)
\item Thread-\/safe operation with atomic state variables
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Each \doxylink{classSimSession}{Sim\+Session} operates independently and can connect to different MSFS instances or use different configuration profiles. 
\end{DoxyNote}
