\doxysubsection{Detailed Description}
Configuration entry defining mapping between MSFS data and \doxylink{namespaceDIS}{DIS} PDUs. 

Each \doxylink{structMappingEntry}{Mapping\+Entry} defines how a specific piece of flight simulation data should be mapped to/from \doxylink{namespaceDIS}{DIS} Protocol Data Units, including rate limiting and enabling/disabling capabilities.
