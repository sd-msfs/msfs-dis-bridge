This repository implements a {\bfseries{\doxylink{namespaceDIS}{DIS} Gateway bridge}} that uses Microsoft Flight Simulator 2024 (MSFS 2024) as a terrain and vehicle renderer for \doxylink{namespaceDIS}{DIS} simulations. It\+:


\begin{DoxyItemize}
\item Parses incoming DIS‑6 PDUs (via the Open‑\+DIS C++ library)
\item Maps \doxylink{namespaceDIS}{DIS} object types to Sim\+Connect variables
\item Writes data into MSFS using the Sim\+Connect SDK
\item Encodes and broadcasts PDUs back onto the network
\item Exposes a REST API for runtime configuration and diagnostics
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{DIS‑6 Only}}\+: Pruned to only support \doxylink{namespaceDIS}{DIS} Protocol Version 6
\item {\bfseries{Multi‑threaded}}\+: One thread per MSFS instance feed for low-\/latency processing
\item {\bfseries{Configurable}}\+: JSON‑based mapping profiles, editable via REST
\item {\bfseries{Vendored SDKs}}\+: MSFS 2024 and Sim\+Connect SDKs under {\ttfamily extern/} for reproducible builds
\item {\bfseries{Test Suites}}\+: PDU replay and Sim\+Connect mock tests for CI integration
