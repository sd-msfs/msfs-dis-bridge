\doxysubsection{Detailed Description}
Singleton UDP multicast sender for \doxylink{namespaceDIS}{DIS} network communication. 

The \doxylink{classUDPMulticaster}{UDPMulticaster} class provides a singleton interface for sending \doxylink{namespaceDIS}{DIS} Protocol Data Units (PDUs) over UDP multicast. It manages a background sender thread and a thread-\/safe queue for efficient asynchronous transmission of PDU data.

This class implements\+:
\begin{DoxyItemize}
\item Singleton pattern for application-\/wide multicast management
\item Asynchronous PDU transmission using a background thread
\item Thread-\/safe enqueueing of PDU data
\item Configurable multicast parameters (group, port, TTL, loopback)
\item Proper socket management and cleanup
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This implementation is Windows-\/specific, using Winsock2 APIs. The class handles Winsock initialization and cleanup internally.
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
As a singleton, this class should not be copied or moved. Access should only be through \doxylink{classUDPMulticaster_a428a56014d20d60fe0c76704e18f0af1}{get\+Instance()}. 
\end{DoxyWarning}

Stop UDP multicast transmission. 

Gracefully shuts down the multicast sender by stopping the background thread, closing the socket, and cleaning up resources. Any remaining queued PDUs will be discarded.
