\doxysubsection{Detailed Description}
Structure containing flight simulation data from MSFS. 

This packed structure represents all the flight data variables that are requested from Microsoft Flight Simulator via Sim\+Connect. The data corresponds to various aircraft parameters that are transmitted over the \doxylink{namespaceDIS}{DIS} network.

\begin{DoxyNote}{Note}
The structure uses pragma pack to ensure consistent memory layout across different compilers and platforms 
\end{DoxyNote}
