\doxysubsection{Detailed Description}
A generic event structure to shuttle data between modules The \doxylink{structInternalEvent}{Internal\+Event} struct provides a standardized way to pass data between different modules in the MSFS \doxylink{namespaceDIS}{DIS} Bridge system. It uses a name-\/based identification system with a flexible payload structure for numeric data. 

Internal event structure for inter-\/module communication.

This structure is designed to be lightweight and efficient for inter-\/module communication. The payload uses double precision floating point numbers to accommodate various types of simulation data while maintaining precision.

\begin{DoxyNote}{Note}
The payload is limited to numeric (double) values for simplicity and performance reasons.
\end{DoxyNote}
This structure represents events that flow between different components of the MSFS \doxylink{namespaceDIS}{DIS} Bridge system, carrying typed data with associated parameters.
