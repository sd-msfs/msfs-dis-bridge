\doxysubsection{Detailed Description}
The \doxylink{classDecode}{Decode} class allows the bridge to tranform incoming PDU packets back into the \doxylink{structFlightData}{Flight\+Data} structure. 

\begin{DoxyWarning}{Warning}
NOT CURRENTLY IMPLEMENTED -\/ This class is for the future implementation of a two way bridge
\end{DoxyWarning}


The \doxylink{classDecode}{Decode} class provides the functionality to tranform incoming \doxylink{namespaceDIS}{DIS} packets back into \doxylink{structFlightData}{Flight\+Data} structures. This would transform the current one-\/way bridge into a two-\/way one.

It is the reverse operation of the \doxylink{classEncode}{Encode} class.
