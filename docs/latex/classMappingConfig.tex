\doxysubsection{Detailed Description}
Central configuration manager for MSFS-\/\+DIS data mapping. 

The \doxylink{classMappingConfig}{Mapping\+Config} class manages the configuration that defines how Microsoft Flight Simulator data is mapped to and from Distributed Interactive Simulation (\doxylink{namespaceDIS}{DIS}) Protocol Data Units (PDUs).

This class serves as the bridge between different data formats, providing conversion methods and maintaining mapping configurations loaded from external files. It supports both event-\/driven and rate-\/limited data transmission.
