\doxysubsection{Detailed Description}
Encoder class that converts \doxylink{structFlightData}{Flight\+Data} structures to \doxylink{namespaceDIS}{DIS} PDU format. 

The \doxylink{classEncode}{Encode} class provides functionality to transfrom flight simulation data from MSFS 2024 into Distributed Interactive Simulation (\doxylink{namespaceDIS}{DIS}) Protocol Data Units (PDUs). This enables DIS-\/compliant simulation systems to interface with MSFS to view simulated flights that are happening in MSFS in real time.

The encoding process involves\+:
\begin{DoxyEnumerate}
\item Converting \doxylink{structFlightData}{Flight\+Data} to an internal event
\item Mapping the event to a \doxylink{namespaceDIS}{DIS} PDU using configurable field mappings
\item Marshaling the PDU into a network-\/ready byte format
\end{DoxyEnumerate}
