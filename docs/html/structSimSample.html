Detailed Description

Structure containing flight simulation data from MSFS.

This packed structure represents all the flight data variables that are requested from Microsoft Flight Simulator via SimConnect. The data corresponds to various aircraft parameters that are transmitted over the DIS network.

Note
The structure uses pragma pack to ensure consistent memory layout across different compilers and platforms
