using IpToIndexFn = std::function<std::optional<int>(const std::string&)>

Function type for mapping client IP addresses to configuration indices.

This function type defines the signature for callbacks that resolve client IP addresses to configuration profile indices. Used to determine which configuration a particular client should use.

Parameters
clientIpThe IP address string of the connecting client
Returns
Optional configuration index, or std::nullopt if no mapping exists
Note
The configuration index should correspond to a valid configuration profile that can be used with SessionManager::addSession()
