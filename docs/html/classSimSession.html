Detailed Description

Individual simulation session managing one SimConnect connection.

The SimSession class represents a single connection to a Microsoft Flight Simulator instance via SimConnect. Each session runs independently in its own thread and manages the full lifecycle of data collection from MSFS.

This class handles:

  • Establishing and maintaining SimConnect connection
  • Defining and requesting flight data variables
  • Processing incoming simulation data
  • Managing simulation state (running/paused)
  • Thread-safe operation with atomic state variables
Note
Each SimSession operates independently and can connect to different MSFS instances or use different configuration profiles.
